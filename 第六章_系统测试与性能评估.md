### 6.1.2 集成测试用例设计

当前集成验证资产包括：

1. `src/cli/app_tests.rs`：通过 `run_from_args(...)` 验证 host/web 参数到输出的端到端链路。
2. `scripts/reverse_smoke.sh`：`backend-status -> analyze -> decompile-run(index)` 冒烟链。
3. `scripts/ci_web_bench_gate.sh`：Web 回归门禁（命中与耗时双约束）。
4. `scripts/reverse_baseline_run.sh`：批样本 reverse 基线统计（`success_rate`、`elapsed_ms_avg`、`elapsed_ms_p95`）。
5. `scripts/web_bench_compare.sh`：`rscan/ffuf/gobuster` 同目标同词典对比。

结论：仓库已具备“库内断言 + CLI 级回归 + 脚本化基线”的三级验证能力。

1. `rscan` 单二进制覆盖 `host + web + vuln + reverse + TUI/Zellij`。
